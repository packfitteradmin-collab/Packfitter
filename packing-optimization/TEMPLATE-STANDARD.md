# PackFitter Page Template Standard
**Last updated: April 2026**

This document defines the permanent section order for every page type on PackFitter.com.
Do not make page-specific judgment calls outside this system.

---

## Type 1: Trip Packing Pages

**Pages:** what-to-pack-3-day-trip, 5-day, 7-day, 10-day, 14-day

**Required section order:**

1. Hero
2. Quick Answer (compact item list + volume range)
3. Quick Decision (bold yes/no + 1-2 supporting sentences)
4. Calculator (full embedded calculator)
5. Packing List (Essentials)
6. Trip Variations
7. Volume Estimate
8. Packing Optimization (compression cubes affiliate)
9. Carry-On Reality (10-day only — optional for others)
10. Constraints
11. Strategy
12. Bag Size Recommendation (Gear)
13. Internal Links
14. FAQ (if present)
15. Bottom Line
16. Notes

**Calculator intro pattern:**
"Most [N]-day trips [fit assessment]. Use this to see if your exact setup actually fits — based on real packing volume."

**Quick Answer pattern:**
- 1 intro sentence
- Compact packing line (pack-qty spans, dot-separated)
- 1 volume range sentence

**Quick Decision pattern:**
- 1 bold yes/no sentence
- 1-2 supporting sentences

## Type 3: Airline Size-Specific Pages

**Pages:** spirit/frontier/ryanair x carry-on-size + personal-item-size

**Required section order:**

1. Hero
2. Quick Answer
3. Definitions / What Counts As
4. What Actually Fits / Real-World Fit
5. Backpack Size Guidance / Size Guide
6. Recommended Bags (product cards + affiliate links)
7. Cluster Links
8. Calculator CTA (link to /will-it-fit.html)
9. FAQ
10. Bottom Line (if present)

**Calculator type:** CTA link only (not embedded)

---

## Type 4: Bag-Fit Pages

**Pages:** {airline}-{size}l-backpack.html (30+ pages)

**Required section order:**

1. Hero (with quick answer box)
2. Fit Verdict / Answer Box
3. Calculator (full embedded calculator)
4. Supporting Analysis
5. FAQ (if present)
6. Bottom Line (if present)

**Calculator type:** Full embedded calculator, positioned immediately after fit verdict

---

## Rules

- Calculator placement is determined by template type, not individual page judgment
- Do NOT move affiliate sections (Packing Optimization, Recommended Bags, Gear)
- Do NOT move internal link blocks
- Quick Answer and Quick Decision must use the same tone, length, and structure across all Type 1 pages
- All volume ranges must match the existing Volume Estimate section on the same page
- This standard supersedes any previous page-specific placement decisions
